Instalación y configuración de Apache2

Saúl Baena Osuna

En esta guía vamos a instalar y configurar Apache2, uno de los servidores web más utilizados en el mundo. Aprenderás cómo ponerlo en marcha en tu sistema, entender su estructura básica y ajustar los elementos esenciales para que puedas alojar tus propios sitios o servicios web. El objetivo es que, al finalizar, tengas un servidor funcional y conozcas los fundamentos necesarios para adaptarlo a tus necesidades, ya sea para proyectos personales, pruebas en un entorno local o despliegues más completos.

Antes de empezar asegurate de que tienes un servidor con una IP fija recomendablemente.



Paso 1 - Instalación y activación del servicio Apache2

En este paso vamos a usar los siguientes comandos para poder actualizar paquetes e instalar apache2:

$ sudo apt update -y

$ sudo apt install apache2 -y

Ahora comprobaremos el estatus del servicio de apache2, lo iniciaremos y lo habilitaremos para que se ejecute automáticamente en el inicio:

$ sudo systemctl status apache2

$ sudo systemctl start apache2

$ sudo systemctl enable apache2

Paso 2 - Creación de host virtual

Vamos a crear una copia del directorio web de ejemplo, en ese directoriose guardará el contenido del servidor web, ahí es donde se pondrían los archivos de nuestra página web:

$ cp -r /var/www/html /var/www/mi-web

Ahora vamos a copiar el fichero de configuración por defecto de apache2 para poder configurar nuestro servidor web:

$ cd /etc/apache2/sites-available

$ cp 000-default.conf mi-server.conf

Entraremos al fichero de configuración del host virtual de apache2 y editaremos el contenido según el fichero de abajo, reemplazando por los datos necesarios, en la guía se usa el puerto por defecto, y datos de ejemplo:

$ nano mi-server.conf 

<VirtualHost *:80>
   # Dirección de correo del administrador/contacto de la web
   ServerAdmin webmaster@localhost
   # Ruta al fichero raíz de la web, si se deja vacío se interpretará que el fichero se llama "index.html"
   DocumentRoot /var/www/mi-web/(nombre del fichero raiz o vacío)
   # En este campo se indicará el nombre de dominio que se va a usar para acceder a la web,
   # si no se pone se tendrá que acceder con la ip especificada en la linea 1, en este caso la ip del server.
   ServerName ejemplo.com
   # Configuraciones extras para el servidor web...
<VirtualHost>

Paso 3 - Habilitar el sitio y aplicar cambios

Antes de acabar hay que habilitar el sitio web para que sea accesible:

$ sudo a2ensite mi-server.conf

Y por último vamos a aplicar todos los cambios, esto se hace reiniciando el servicio de apache2, y si todo ha salido bien ya tendremos listo nuestro servidor web:

$ sudo systemctl restart apache2

Con esto nuestra web ya debería de estar accesible a través del navegador web, accede desde del dominio o IP según tu configuración.
